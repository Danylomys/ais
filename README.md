# Репозиторій студента Мисишина Данила Петровича

Цей репозиторій створений для виконання лабораторних робіт з дисципліни **"Адміністрування інформаційних систем"** (DevOps-напрям).

## Призначення
Репозиторій використовується для підготовки робочого середовища, налаштування інструментів DevOps, а також виконання лабораторних робіт, зокрема:
- налаштування Git;
- підготовка Docker-середовища та створення Dockerfile;
- створення структури Terraform;
- організація файлової структури проєкту.

## Структура репозиторію
- **ansible/** — директорія для роботи з ansible
- **docker/** — директорія для роботи з Docker, містить `Dockerfile` та повʼязані конфігурації.
- **terraform/** — початкова структура для Terraform-конфігурацій.
- **README.md** — інформація про репозиторій та його призначення.

## Дисципліна
Лабораторні роботи виконуються з предмету **"Адміністрування інформаційних систем"**, Національний університет «Львівська політехніка».
